# Projeto Nega - Sistema de Gestão de Pedidos

Sistema em desenvolvimento voltado para gerenciamento de pedidos e organização de dados.

## 🚀 Tecnologias
- HTML5
- CSS3
- JavaScript

## 🔄 Em evolução
- Integração com React
- Estruturação com SQL (banco de dados)

## 🎯 Objetivo
Criar uma aplicação prática para:
- Cadastro de pedidos
- Organização de informações
- Estrutura escalável para banco de dados

## 📌 Status
Em desenvolvimento.
